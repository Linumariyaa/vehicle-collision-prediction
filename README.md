Vehicle Collision Prediction System

This project predicts potential vehicle collision risk based on speed and distance between vehicles.

Features
- Predicts collision risk
- Simple Python logic
- Easy to understand algorithm
- Useful for intelligent transportation systems

Technologies Used
- Python
- Pandas
- Machine Learning (Scikit-learn)

How to Run

1. Install Python
2. Install required libraries:
   pip install pandas matplotlib scikit-learn

3. Run the program:
   python collision_prediction.py
