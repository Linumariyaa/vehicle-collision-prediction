# Vehicle Collision Prediction System

A Python-based project that predicts potential vehicle collision risk based on vehicle speed and distance using Machine Learning.

## Features
- Predicts collision risk using a Decision Tree model
- Data-driven approach using a sample dataset
- Easy-to-understand Python logic
- Includes data visualization for speed vs distance
- Can be extended for intelligent transportation systems

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn (Machine Learning)
